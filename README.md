# QuizWhiz+

A feature-rich quiz web application built with HTML, CSS, and Vanilla JavaScript.

## Features
- Multiple question categories
- Difficulty levels
- Score tracking
- Leaderboard system
- Answer review
- Dark/Light theme
- Responsive design

## Installation
1. Clone the repository
2. Open `public/index.html` in a browser
